# Salyut-J
Salyut-J是本人为完成北航计算机学院25年秋季学期“编译技术”课程的课程设计，而使用Java编写的玩具编译器，支持课程组于当年修改的SysY语言文法。

其名来自于“礼炮”空间站计划。原计划使用 C++ 编写并命名为 "Salyut-C"，后因本人薄弱的 C++ 基础难以支撑整个项目的开发，而转向使用 Java 开发，并更名整个项目为 "Salyut-J"。

main 分支存放了本人的设计源码，docs 分支存放了本人应课程要求撰写的设计文档、优化文章和总结感想，以供阁下参考。

本编译器的设计，在课程开展的性能排名中总排 4/310, 不过这个排名于课程之外也没有什么用了。具体实现了什么优化，还请移步文档部分的优化文章。
